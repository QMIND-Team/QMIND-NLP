How to run this:

1. Get a copy of all of the recordings, in wav form
2. Extract them into "WavFiles"
3. Run AddAccentColumn.py
4. Move the newly created speakers_all_new.csv into WavFiles
5. In WavFiles, run deleteUnneeded.py
6. Run MakeMFCCCSV.py
7. Run decodeCSV.py
8. Wait a long time and cry as all of your ram is consumed
9. Admire the results!
